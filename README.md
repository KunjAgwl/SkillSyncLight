### 💥 WHAT IS JOBFINDER AI?
You are a brave job seeker equipped with a PDF. Your quest? Find the listing that perfectly matches your XP. **JobFinder AI** is your semantic radar, scanning your skills using **TF-IDF NLP algorithms** to locate the exact job coordinates where you stand a 99% match rate. 

Forget manually searching. Drop the PDF, hit start, and let the algorithm do the grinding.

---

### 🚀 OVERPOWERED FEATURES

<kbd>📄 <b>AUTO-SCAN</b></kbd> : Extracts raw stats (skills & experience) from your PDF Resume in milliseconds using `pdf-parse`. <br><br>
<kbd>🧠 <b>TF-IDF BRAIN</b></kbd> : Evaluates your loadout against a JSON array of job listings. Direct hits are boosted instantly.<br><br>
<kbd>🔮 <b>GLASS UI</b></kbd> : A polished, neon-tinted glassmorphic HUD so you can view your matches in absolute style.<br><br>
<kbd>💾 <b>NO-DB SETUP</b></kbd> : Runs completely locally off flat JSON files. No heavy Postgres databases to fight just to get it running.

### 👾 HOW TO INSTALL & PLAY

**1. Clone the Cartridge**
```bash
git clone <your-repo-url>
cd Dumb2
```

**2. Power Up (Install Node Modules)**
```bash
npm install express multer pdf-parse cors natural
```

**3. Boot the Server**
```bash
node server.js
```

**4. Access the Terminal**
Warp to `http://localhost:5000` via your web browser to enter the grid grid.
