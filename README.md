## 🕹️ PUSH START: PROJECT OVERVIEW

**JobFinder AI** is a lightweight, blazing-fast web app that matches your resume to jobs using semantic AI magic. No bloated databases, no complex setup. Just drop your PDF resume into the machine, and let the TF-IDF matching engine reveal your next career move!

---

## 🌟 GAME FEATURES

- **`[ 📄 ]` PDF Scanning Magic**: Extracts your skills and experience straight from a PDF resume using `pdf-parse`.
- **`[ 🧠 ]` TF-IDF Brain**: Analyzes the semantic similarity between your resume and a JSON database of job listings using natural language processing.
- **`[ 🔮 ]` Crystal Clear UI**: A stunning glassmorphic interface with interactive animations and satisfying hover effects. 
- **`[ ⚡ ]` Zero-Config Server**: Stores users and jobs natively in local flat JSON files for ultimate portability and rapid local development.

## 🎮 HOW TO PLAY (Installation)

1. **Clone the Source Code**
   ```bash
   git clone <your-repo-url>
   cd Dumb2
   ```

2. **Equip Your Items (Install Dependencies)**
   ```bash
   npm install
   ```
   *(Required items: `express`, `multer`, `pdf-parse`, `cors`, `natural`)*

3. **Start the Engine!**
   ```bash
   node server.js
   ```

4. **Enter the Portal**
   Open your browser and navigate to [http://localhost:5000](http://localhost:5000)

## 🗺️ LEVEL DESIGN (Architecture)

```text
📁 public/         --> The Client-Side Realm (HTML, CSS, Vanilla JS)
📄 matcher.js      --> The AI Brain (Skill Extraction & Semantic Scoring)
📄 server.js       --> The Grand Controller (Express Routing & File Uploads)
📄 jobs.json       --> The Loot Drops (Mock Database of Job Listings)
📄 users.json      --> The Save File (Local User Data Storage)
```

---

## ⚙️ SYSTEM REQUIREMENTS

- **Node.js**: v14.0 or higher
- **Browser**: Any modern browser that can handle CSS Grid, Flexbox, and backdrop-filter (for that sweet glassmorphism).
